## Käyttö

Ohjelma käännetään make:lla. Make-skripti käyttää gcc:tä kääntäjänä.

```console
foo@bar:~$ make
```
Make-skripti luo ajettavan ohjelman wish, joka ajetaan seuraavasti:

```console
foo@bar:~$ ./wish
```
Ohjelma ajetaan batch-tilassa seuraavasti:

```console
foo@bar:~$ ./wish <tiedostonimi>
```
